🔄 Data Flow
Engines (business logic) → pure Python, no Streamlit
Pages (UI) → call engines, render with st.*
Components → reusable widgets used by pages
Router → dispatches to correct page
Theme → injected once, applies everywhere
📱 Share Flow
Any page → share_button component
→ Format data (summary/table/chart image)
→ Choose: WhatsApp / Telegram / Email / PDF
→ share_system.py handles delivery
→ Logs to share_history for tracking
⚡ Key Point
Existing command_intel/ files aur engines mostly as-is rahenge. Sirf import paths update honge. Naya code = router, theme, components, share system, telegram engine, aur skeleton features ko complete karna.
