✅ Top Bar Rules
• 7 modules visible always
• "More ▾" = Compliance + System & AI
• Active module = blue highlight
• Right side: WA, TG, Notifications, Search
✅ Sidebar Rules
• Shows features of active module
• Active feature = blue left border
• ★ marks starred features
• Quick Actions section at bottom
• Always visible (not collapsible)
✅ Content Rules
• Title + breadcrumb at top
• Filter + Share buttons top-right
• Full width content below
• Per-page local filters
• Share button on every page
