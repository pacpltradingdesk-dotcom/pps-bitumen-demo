# 🚀 Bitumen Sales Dashboard

A comprehensive sales management system for bitumen trading built with Streamlit.

## Features
- 📊 **Pricing Calculator** - Calculate landed costs with real-time freight
- 💼 **Sales Workspace** - Deal structuring and margin analysis
- 🏭 **Feasibility Assessment** - Compare refineries, imports, decanters
- 📅 **Sales Calendar** - Season & holiday planning
- 🎯 **CRM & Tasks** - Customer relationship management
- 🚨 **SOS Alerts** - Price drop notifications
- 📤 **Reports** - Sales analytics and charts

## Tech Stack
- **Frontend:** Streamlit
- **Data:** Pandas, NumPy
- **PDF:** ReportLab, FPDF2
- **Market Data:** yFinance

## Quick Start
```bash
pip install -r requirements.txt
streamlit run dashboard.py
```
